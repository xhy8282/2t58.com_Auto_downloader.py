# 2t58.com 音乐批量下载器

一个用于批量下载 2t58.com 网站歌手音乐的 Python 爬虫脚本。

## ⚠️ 项目状态

**此项目已不再维护，且目标网站可能已失效。**

- 最后修改日期：2024年2月17日
- 当前状态：不可用
- 更新计划：不确定

本项目仅作为早期学习作品存档，供参考之用。

## 功能特性

- 批量下载指定歌手的所有音乐
- 多线程并发下载，提升效率
- 支持断点续传，避免重复下载
- 自动文件名规范化处理
- 智能 URL 路径纠正

## 使用方法

### 环境要求

```bash
pip install requests beautifulsoup4 html5lib
```

### 运行方式

**交互式运行：**
```bash
python 复杂2_网络爬虫.py
```

**命令行参数：**
```bash
python 复杂2_网络爬虫.py [下载路径] [歌手页面URL] [线程数]
```

### 使用示例

1. 运行脚本后，输入歌手页面 URL（如 `http://www.2t58.com/singer/xxxxx.html`）
2. 指定下载保存路径
3. 设置下载线程数
4. 脚本将自动检索并下载该歌手的所有音乐

## 技术实现

- **爬虫框架**：BeautifulSoup4 + html5lib
- **并发下载**：ThreadPoolExecutor 多线程池
- **断点续传**：本地 record.txt 记录下载进度
- **API 解析**：通过 POST 请求获取真实 MP3 下载链接
